Implementation Notes

Overview

This Google Docs add-on provides tools for assisting with patent document drafting. The tools automate formatting, reference insertion, feature extraction, and text normalization tasks.

Formatting Tools

  • Apply predefined font colors (Blue / Gray)
  • Convert text to Title Case
  • Convert text to lower case

Reference Insertion

The reference tool supports:

  • Paragraph references
  • Page references
  • Abstract references

Example output:

(see paragraph [0025])

Date Tool

  • Insert today's date
  • Insert a manually entered date

Feature Extraction

Generates formatted feature text from structured document sections.

Limitations

  • Tools require text selection in most cases.
  • Formatting tools work only on text elements.
  • Images and drawings are not supported.
